## Task Overview
This repository contains my work for Task 4 of the Prodigy InfoTech Data Science Internship. The objective of this task was to analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes.

## Tools and Libraries Used
The analysis was performed using Google Colab and Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

## Data Preparation
The dataset was cleaned by handling missing text values and preparing the data for sentiment analysis. Text data was processed to enable meaningful visualization and analysis.

## Sentiment Analysis
Sentiment distribution was analyzed to understand the overall polarity of social media posts. Additional visualizations were created to study sentiment trends across different topics.

## Conclusion
